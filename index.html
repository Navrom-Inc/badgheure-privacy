Politique de Confidentialité

Dernière mise à jour : 4 mars 2026

Badg'Heure est une application de suivi du temps de travail développée par Navrom Inc. Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons vos informations.

1. Données collectées

Badg'Heure collecte les données suivantes :

  • Données de badgeage : heures d'arrivée, de départ, de pause. Ces données sont stockées localement sur votre appareil.
  • Informations de compte Google : si vous choisissez de vous connecter avec Google, nous accédons à votre nom, adresse e-mail et photo de profil pour l'authentification.
  • Données de synchronisation : si vous activez la synchronisation cloud, vos données de badgeage sont stockées de manière sécurisée sur Firebase (Google Cloud).
  • Achats intégrés : les informations relatives à vos achats sont gérées par Google Play. Nous stockons uniquement le statut d'activation de vos fonctionnalités premium (validé côté serveur via Firebase Cloud Functions).

2. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Afficher vos heures de travail, rapports et statistiques.
  • Synchroniser vos données entre vos appareils (si activé).
  • Générer des exports CSV de vos badgeages.
  • Valider vos achats intégrés côté serveur pour garantir leur authenticité.

Nous ne vendons, ne partageons et ne transmettons aucune donnée personnelle à des tiers.

3. Stockage et sécurité

  • Stockage local : par défaut, toutes les données restent sur votre appareil via AsyncStorage.
  • Stockage cloud : si la synchronisation est activée, vos données sont stockées sur Firebase Firestore, protégées par des règles de sécurité qui isolent vos données (seul vous pouvez y accéder).
  • Traitement serveur : les achats intégrés sont validés côté serveur via Firebase Cloud Functions. Aucune donnée personnelle n'est transmise lors de cette validation, seuls le jeton d'achat et l'identifiant du produit sont vérifiés.
  • Obfuscation : le code de l'application est compilé en bytecode et obfusqué pour protéger la propriété intellectuelle.

4. Services tiers

L'application utilise les services tiers suivants :

  • Firebase Authentication : pour la connexion Google — Politique Firebase
  • Firebase Firestore : pour la synchronisation cloud — Conditions Google Cloud
  • Firebase Cloud Functions : pour la validation des achats côté serveur — Conditions Google Cloud
  • Google Play Billing : pour les achats intégrés — Politique Google

5. Droits des utilisateurs (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous pouvez à tout moment :

  • Accéder à vos données via l'historique et les rapports de l'application.
  • Exporter vos données au format CSV depuis les paramètres.
  • Supprimer vos données locales en désinstallant l'application.
  • Supprimer vos données cloud en vous déconnectant de votre compte dans les paramètres.
  • Demander la suppression totale de toutes vos données en nous contactant par e-mail.

6. Publicité

Badg'Heure ne contient aucune publicité et n'utilise aucun SDK publicitaire.

7. Mineurs

L'application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants.

8. Modifications

Nous nous réservons le droit de modifier cette politique. Toute modification sera publiée sur cette page avec une date de mise à jour.
